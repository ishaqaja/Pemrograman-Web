Tentang CSS

Ini adalah paragraf di dalam sebuah div dengan class "highlight". Class ini memberikan latar belakang putih, padding, sudut yang membulat, dan efek bayangan (box-shadow). Paragraf ini juga menggunakan styling default untuk elemen 'p'.

Informasi Penting

Ini adalah kotak informasi yang menggunakan class "info-box". Perhatikan latar belakang biru muda dan border birunya. Heading H3 di dalamnya berwarna oranye.

Daftar Hal yang Dipelajari

  • Elemen Heading (h1, h2, h3)
  • Elemen Paragraf (p)
  • Selector Class (.highlight, .info-box)
  • Styling untuk Unordered List (ul) dan List Item (li)
